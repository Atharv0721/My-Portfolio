# Atharv Gupta Portfolio

This is my personal developer portfolio website built to showcase my projects, technical skills, and achievements. The website highlights my work in full-stack development and innovative technologies through an interactive and modern user interface.

## 🚀 Tech Stack

- React
- TailwindCSS
- Vite
- JavaScript


## ✨ Key Features

- ⚡ Modern and responsive UI design
- 🌙 Light / Dark theme toggle
- 🌍 Multilingual language support
- 📱 Fully responsive across devices
- 🎨 Smooth animations and interactive elements
- 🧩 Clean and modular component architecture
- 📂 Project showcase with detailed descriptions
- 📬 Functional contact form
- 🔗 Social media integration

## 📂 Featured Projects

- **InterviewIQ – AI Mock Interview Platform**  
  AI-powered platform that simulates technical interviews and provides intelligent feedback.

- **HerbTrace – Smart IoT Herb Storage System**  
  IoT-based system for monitoring herb storage conditions with blockchain traceability.

- **Crowdsourced Traffic Management System**  
  A real-time traffic monitoring system using Google Maps API that allows users to report and view traffic updates.


## 🛠️ How to Run This Project Locally

1️⃣ Clone the Repository
2️⃣ Navigate to the Project Folder
3️⃣ npm install
4️⃣ npm run dev
5️⃣ Open in Browser
After running the server, Vite will show something like:

Local: http://localhost:5173/

Your portfolio website will start running locally.
